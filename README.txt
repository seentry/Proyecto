TITLE: Alicia Estetica

Objective
The objective of this project is to develop a web platform for an aesthetic clinic that enables patients to easily access information about treatments and services, as well as manage their appointments online. The application addresses common issues such as unclear communication and difficulties in scheduling consultations, ultimately enhancing user experience and optimizing clinic operations.


Key Features
Appointment Scheduling: Allows users to book, reschedule, and cancel appointments intuitively.
Service Information: Provides detailed descriptions of treatments, procedures, and aesthetic services.
Responsive Design: Optimized for desktops, tablets, and mobile devices to ensure an optimal user experience.
Before & After Gallery: Showcases treatment results with high-quality images.
Contact and Support: Features an integrated contact form for inquiries and customer service.

Technologies Used
Languages: PHP, JavaScript, HTML, and CSS.
Frameworks and Libraries: Symfony, Angular, and Bootstrap.


Launch Requirements
Hardware: Any device capable of running a modern web browser and open de server.
